      • setReuseKeys

        public void setReuseKeys(boolean reuse)
        Determines whether the public and private key generated for the CA will be reused for other hosts as well. This is mostly just a performance optimisation, to save time generating a key pair for each host. Paranoid clients may have an issue with this, in theory.
        Parameters:
        reuse - true to reuse the CA key pair, false to generate a new key pair for each host
