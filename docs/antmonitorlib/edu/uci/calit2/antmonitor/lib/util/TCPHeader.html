Class TCPHeader

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.util.TCPHeader

  • class TCPHeader
extends java.lang.Object
    This class represents a TCP Header. Can be used to parse a TCP Header. Note that as of now the header flags are simply parsed as a whole and are not distinguished individually.
