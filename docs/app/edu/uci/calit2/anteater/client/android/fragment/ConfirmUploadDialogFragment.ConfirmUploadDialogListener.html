Interface ConfirmUploadDialogFragment.ConfirmUploadDialogListener

  • Enclosing class:
    ConfirmUploadDialogFragment

    public static interface ConfirmUploadDialogFragment.ConfirmUploadDialogListener
    Callback interface. Used by ConfirmUploadDialogFragment to deliver information to its host. Any activity hosting a ConfirmUploadDialogFragment must implement this interface (if not, a crash will occur during ConfirmUploadDialogFragment.onAttach(android.app.Activity)).

      • onUploadConfirmed

        void onUploadConfirmed(ConfirmUploadDialogFragment dialogFragment,
                       java.io.File[] filesToUpload)
        Invoked when the user confirms to start the file upload.
        Parameters:
        dialogFragment - The ConfirmUploadDialogFragment that invoked this method.
        filesToUpload - The set of files that should be uploaded (i.e. the set of files that has the user's approval). Note that this might be a subset of TrafficLogFiles.getCompleted(android.content.Context) as the set of completed files may have increased if there is heavy data traffic in the background while waiting for the user to confirm the upload. As such you should only upload filesToUpload and not the super set exposed by TrafficLogFiles.getCompleted(android.content.Context) .
