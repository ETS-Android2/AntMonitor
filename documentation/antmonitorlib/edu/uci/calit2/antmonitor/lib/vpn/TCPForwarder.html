Class TCPForwarder

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.vpn.TCPForwarder

  • class TCPForwarder
extends java.lang.Object
    Maintains an external socket connection to send and receive TCP data. One TCP forwarder is created for one TCP connection made by an app.
