Class WifiAndPowerConnectivityReceiver

  • java.lang.Object
    • android.content.BroadcastReceiver
      • edu.uci.calit2.anteater.client.android.device.WifiAndPowerConnectivityReceiver
  • Direct Known Subclasses:
    PassiveWifiAndPowerConnectivityReceiver

    public abstract class WifiAndPowerConnectivityReceiver
extends android.content.BroadcastReceiver
    Abstract receiver that watches for WiFi and power supply connectivity. Subclasses should implement onPowerAndOnWifi(Context context) to react to power and WiFi being connected.

    If an Intent with action equal to Intent.ACTION_POWER_CONNECTED is received, this receiver queries the charging state. If the device is also charging, this receiver invokes onPowerAndOnWifi(Context context) in order to let subclass implementations react to the device being connected to WiFi and power.

    If an Intent with action equal to WifiManager.NETWORK_STATE_CHANGED_ACTION is received, this receiver checks if the new WiFi state is the connected state and if power is also connected. If WiFi and power are both connected, this receiver invokes onPowerAndOnWifi(Context context) in order to let subclass implementations react to the device being connected to WiFi and power.
